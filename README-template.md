### What I learned

CSS Grid Layout

```css
  .violet {
    grid-column: 1/3;
  }

  .blackish-blue {
    grid-column: 2/4;
  }

  .container .white:last-of-type {
    grid-column: 4;
    grid-row: 1/3;
  }
```
